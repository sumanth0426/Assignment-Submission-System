# **App Name**: AssignPro

## Core Features:

- Admin Dashboard: Create and manage faculty and student accounts, academic years, semesters, branches, sections, and subjects. Manage co-admins.
- Faculty Assignment Management: Assign students to courses, create assignments with deadlines, and view student submissions.
- Student Dashboard: View assigned assignments, upload submissions as PDFs, and track verification status (pending, verified, rejected).
- Submission Deadline Enforcement: Automatically prevent submissions after the deadline. Display status to students, guiding them to contact faculty if needed.
- Document Storage: Securely store student assignments as PDFs using Firestore.
- User Authentication: Secure login for admins, faculty, and students using roll number/username and password stored in Firestore.
- AI-Powered Feedback Tool: Provide AI-generated feedback to faculty about potential areas for improvement, after analyzing a faculty's grading and remarks. Use it as a tool for giving more effective feedback, improving assignment clarity, and spotting potential inconsistencies. Faculty has the option to accept, edit, or discard.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust, knowledge, and professionalism, reflecting the college's engineering focus.
- Background color: Very light blue (#E8EAF6), almost white, to create a clean and modern look.
- Accent color: Purple (#7E57C2), slightly analogous to the primary, to highlight important actions and information.
- Body and headline font: 'Inter' sans-serif for a clean, modern, and easily readable appearance. 
- Use Material Design icons for a consistent and modern look. Icons should be clear and intuitive.
- Employ a clean, well-structured layout with a focus on ease of navigation and a professional aesthetic.
- Subtle animations and transitions to enhance user experience and provide feedback on interactions.
